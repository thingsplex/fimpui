Api

Registry API :

GET /fimp/api/registry/things - List of things from regitry

GET /fimp/api/registry/services - List of services from regitry

GET /fimp/api/registry/interfaces - List of interfaces from regitry

GET /fimp/api/registry/locations - List of locations from regitry

GET /fimp/api/registry/thing/:tech/:address - Thing by address and technology

GET /fimp/api/registry/clear_all - Clear registry

PUT /fimp/api/registry/thing - Upsert thing

Vinculum API :

GET /fimp/vinculum/devices - Get list of devices from Core

GET /fimp/api/vinculum/areas - Get list of areas from Core

GET /fimp/api/vinculum/shortcuts - Get list of shortcuts from Core

GET /fimp/vinculum/import_to_registry - Import devices and locations from CORE to FIMP registry

Flow API :

GET /fimp/flow/list - Get list of Flows

GET /fimp/flow/definition/:id - Get single Flow definition by Id

GET /fimp/flow/context/:id - Get Flow flow context by Flow Id

POST /fimp/flow/definition/:id - Upsert Flow definition

DELETE /fimp/flow/definition/:id - Delete Flow

Global settings

GET /fimp/api/configs - Get global configs

GET /fimp/fr/upload-log-snapshot - Upload log snapshots to cloud storage
